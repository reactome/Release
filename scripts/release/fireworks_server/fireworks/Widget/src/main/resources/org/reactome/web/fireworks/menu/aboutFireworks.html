What is this?
A genome-wide, hierarchical visualisation of Reactome pathways in a space-filling graph.
What is the aim?
Visualisation of genome-scale pathway analysis results, allowing both a quick “first glance” and rapid zooming in on interesting results.

The pathways overview offers an intuitive overview of dataset analysis results in the context of the Reactome hierarchical pathway structure, allowing both a quick “first glance” and rapid zooming in on interesting results.
